# Python Tic-Tac-Toe 🎮

A highly interactive, command-line based Tic-Tac-Toe game developed in Python. This project serves as a demonstration of fundamental game loop mechanics, state management, and algorithmic thinking in Python.

## ✨ Features
- **Interactive Gameplay**: Two-player turn-based system.
- **Input Validation**: Prevents users from overwriting grid spaces or inputting invalid commands.
- **Win Detection**: Algorithms successfully check horizontally, vertically, and diagonally for win states.
- **Draw Detection**: Accurately recognizes when the board is full with no winner.

## 🛠️ Built With
- **Python 3**

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/King-Austin/Tic_Tac_Toe-game.git
   ```
2. Navigate into the directory and run the script:
   ```bash
   python main.py
   ```
